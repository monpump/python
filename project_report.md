# Python 학습 프로젝트 분석 보고서

## 1. 프로젝트 구조

프로젝트는 다음과 같은 주요 디렉토리로 구성되어 있습니다:

```
📁 python_src/
├── 📁 class/           # 클래스 관련 학습
├── 📁 my_test/        # 테스트 코드
├── 📁 try/            # 예외 처리 학습
├── 📁 tuple_set_dic/  # 자료구조 학습
└── 📁 .vscode/        # VS Code 설정
```

## 2. 주요 학습 내용

### 2.1 클래스 (class/)
- 기본 클래스 정의 및 생성자
- 인스턴스 변수와 클래스 변수
- 특수 메서드 (`__init__`, `__str__`, 등)
- 게임 구현 (가위바위보, 단어 맞추기)
- Property 데코레이터를 활용한 접근 제어

### 2.2 자료구조 (tuple_set_dic/)
- 튜플 기초 및 리스트 변환
- 집합(Set) 연산과 메서드
- 딕셔너리 CRUD 연산
- 데이터 정렬과 필터링
- 컬렉션 관련 함수 (map, zip, enumerate)

### 2.3 예외 처리 (try/)
- try-except 구문
- 다양한 예외 타입 처리
- 사용자 정의 예외
- 게임 로직에서의 예외 처리

### 2.4 테스트 (my_test/)
- 기본 테스트 케이스
- 데이터 구조 활용 테스트
